Common Abbreviations

A sigla HTML é a linguagem de programção padrão para criar pagina webs.

A sigla CSS utilizamos para estilizar paginas Web .

A sigla URL é o endereço único de um recurso na internet, como uma página web, imagem ou vídeo, que você digita no navegador para aceder a ele.

A sigla API é um conjunto de regras e protocolos que permite a comunicação entre diferentes sistemas e aplicações.
